-
![snapshot](https://raw.githubusercontent.com/JuncoJet/getSS-ShadowSocks-/master/snapshot.gif)

-

* How to use it

You must install `BeautifulSoup` and `requests` first.

```
pip install BeautifulSoup4
pip install requests
```
